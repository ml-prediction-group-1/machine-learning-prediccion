# Código de Conducta del Proyecto

Este proyecto sigue principios de respeto, colaboración y profesionalidad entre todos los participantes.

## Comportamiento esperado
- Comunicación clara y respetuosa.
- Apertura a opiniones diversas.
- Uso adecuado del repositorio y canales de comunicación.

## Comportamiento inaceptable
- Insultos, ataques personales o descalificaciones.
- Spam o uso inadecuado del repositorio.
- Sabotaje o eliminación de trabajo de compañeros.

## Aplicación
Cualquier incumplimiento podrá ser revisado por los responsables del grupo o el docente.
